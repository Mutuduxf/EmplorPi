# Accounting Rules

- Treat unmatched transactions, invalid dates, invalid amounts, duplicates, and out-of-balance ledgers as review items.
- Do not infer fraud from anomalies; describe them as items requiring human review unless the source evidence proves otherwise.
- For debit/credit ledgers, report total debits, total credits, and the difference.
- For signed-amount exports, document whether expenses are represented as positive or negative values.
- Keep generated reports separate from raw statements, backups, and source ledgers.
