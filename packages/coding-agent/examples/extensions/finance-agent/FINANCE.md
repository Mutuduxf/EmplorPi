# Finance Agent Guidance

Use this context when working with finance files, ledgers, transaction exports, invoices, statements, or generated reports.

- Prefer read-only analysis and preserve original finance records.
- Before editing finance files, summarize the intended impact and request confirmation where possible.
- State calculation sources, columns, filters, and assumptions.
- Cite evidence paths and line numbers for anomalies, reconciliation differences, and data-quality issues.
- Mask sensitive account identifiers and avoid exposing credentials, tokens, or private keys.
- Do not provide investment, legal, audit, or tax advice.
