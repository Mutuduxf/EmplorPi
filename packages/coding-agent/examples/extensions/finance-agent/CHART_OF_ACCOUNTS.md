# Chart of Accounts Context

If a project-specific chart of accounts is not provided, ask the user for the account mapping before reclassifying transactions.

Common grouping hints:

- Revenue: sales, fees, service income, interest income
- Cost of sales: direct materials, fulfillment, merchant fees
- Operating expenses: payroll, rent, software, travel, marketing, utilities
- Assets: cash, bank, receivables, inventory, prepaid expenses
- Liabilities: payables, loans, taxes payable, deferred revenue
- Equity: owner contributions, retained earnings
